# warm-editorial-theme

> **Summary**: The blog uses a warm editorial design theme instead of cold monochrome. Light mode: cream background (#faf8f5), warm brown text (#2c2420), gold primary (#8b6914). Dark mode: warm near-black (#1a1614), cream text (#f5f0e8), gold primary (#e8c47c). Headings use Lora serif (font-heading CSS var), body uses Geist Sans. Badge hovers use warm gold (hover:bg-primary/10 hover:text-primary). Post cards default to capybara-256.webp when no cover image. Applied to: globals.css (CSS custom props), layout.tsx (Lora font import), all page headings (font-heading class), PostCard, TrendingThisWeek, TopicSpotlight, RelatedPosts.
