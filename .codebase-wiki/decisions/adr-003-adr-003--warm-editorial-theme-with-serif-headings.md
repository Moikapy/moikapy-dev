# ADR-003: ADR-003: Warm editorial theme with serif headings

> **Status**: Accepted

## Context
The blog's original theme used cold monochrome oklch colors and all-sans-serif fonts (Geist Sans only). This felt generic and lacked personality for a personal blog.

## Decision
Switched to a warm editorial theme with warm hex colors (cream/brown/gold palette), Lora serif for headings (font-heading CSS var), and gold accent hovers. Dark mode uses warm near-black (#1a1614) with cream text (#f5f0e8) and gold primary (#e8c47c). Badge/tag hover states use primary gold instead of secondary gray. PostCard defaults to capybara-256.webp when no cover image is set.

## Alternatives Considered
Staying with cold monochrome; using a full design system like Tailwind UI; using CSS-in-JS theming
