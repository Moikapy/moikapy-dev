# post-card

> **Summary**: Blog post card component. Shows title, date, reading time, excerpt, and tags. Cover image uses capybara-256.webp as default fallback when no coverImage is set (centered with padding on warm bg-muted). Title uses font-heading (Lora serif). Badges use warm gold hover (primary/10). Card has hover:border-primary/50 and hover:shadow-md transition.
