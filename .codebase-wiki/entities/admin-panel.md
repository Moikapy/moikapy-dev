# admin-panel

> **Summary**: Admin panel at /admin for creating, editing, and deleting blog posts. Cookie-based auth with SHA-256 password hash. Tiptap rich text editor with Markdown support, AI title/slug/excerpt suggestions via GLM-5.1, and ElevenLabs Scribe v2 real-time speech-to-text dictation.
