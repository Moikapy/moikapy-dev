# x402-monetization

> **Summary**: HTTP 402 payment-required protocol for API access. External consumers pay per request (USDC on Base) to access /api/posts and /api/knowledge endpoints. Uses @x402/core, @x402/evm, @x402/next packages. Internal admin access is free via cookie auth.
