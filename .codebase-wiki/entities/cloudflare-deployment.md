# cloudflare-deployment

> **Summary**: Deployed as a Cloudflare Worker via OpenNext (opennextjs-cloudflare). D1 database for blog posts. Secrets managed via wrangler secret: ADMIN_PASSWORD_HASH, SESSION_SECRET, OLLAMA_API_KEY, ELEVENLABS_API_KEY. Static assets served from Workers Sites. wrangler.toml defines bindings for DB and WORKER_SELF_REFERENCE.
