# tiptap-editor

> **Summary**: Rich text editor component built on Tiptap with StarterKit, Link, Image, Placeholder, CodeBlockLowlight, HorizontalRule, and Markdown extensions. Renders as Markdown on update. Toolbar provides undo/redo, formatting, headings, lists, code blocks, blockquote, link/image insertion, and optional voice dictation mic button. Dictation inserts text at cursor via editor.chain().focus().insertContent().
